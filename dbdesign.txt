1. Users Table (Employee, Manager)
This table will store basic user details and roles (either staff or manager).

Column Name	        Data Type	            Description
user_id	INT         (PK)	                Unique ID for the user
user_name	        VARCHAR	                Username for login
password	        VARCHAR	                Encoded password
first_name	        VARCHAR	                First name of the user
last_name	        VARCHAR	                Last name of the user
role	            ENUM (Staff, Manager)	Role of the user
employee_id	INT     (FK to Employees)	    Employee-specific details
status	            VARCHAR	                Account status (active/inactive)


2. Employee Table
This table will hold specific information for staff and managers, such as employee ID and salary.

Column Name	Data    Type	        Description
employee_id	INT     (PK)	        Unique ID for the employee
work_section	    VARCHAR	        Section where the employee works
salary	            DECIMAL	        Salary amount
work_hours	        DECIMAL	        Total work hours


3. Dishes Table
This table manages the menu items and their prices.

Column Name	        Data Type	    Description
dish_id	INT         (PK)	        Unique ID for the dish
dish_name	        VARCHAR	        Name of the dish
category	        VARCHAR	        Category/type of the dish
price	            DECIMAL	        Price of the dish
availability	    BOOLEAN	        Is the dish available?


6. Statistics Table
This table stores revenue and productivity-related statistics.

Column Name	Data    Type	                Description
stat_id	            INT (PK)	            Unique ID for the statistic
employee_id	        INT (FK to Employees)	Reference to the employee
revenue_generated	DECIMAL	                Revenue generated by the employee
total_hours_worked	DECIMAL	                Total hours worked by the employee
date	            DATETIME	            Date of the statistics record


7. Attendance Table
This table stores information about employee login sessions.

Column Name	Data Type	Description
attendance_id	INT (PK)	Unique ID for the attendance record
employee_id	INT (FK to Employees)	Reference to the employee
login_time	DATETIME	Time of login
logout_time	DATETIME	Time of logout


8. Feedback Table
This table manages customer feedback on dishes or service.

Column Name	    Data Type	            Description
feedback_id	    INT (PK)	            Unique ID for the feedback
customer_name	VARCHAR	                Name of the customer giving feedback
feedback_type	ENUM (Food, Service)	Type of feedback
message	        TEXT	                Feedback message
rating	        INT	                    Rating (1-5)
date	        DATETIME	            Date and time of the feedback
